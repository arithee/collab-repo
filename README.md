<h1>Project Description</h1>
<h2>Cozy Cup Cafe is a test website to test out collaborating on Github for our course CC-6 (Application Development and Emerging Technologies 1). I, and John Kristoffer Chiong are the people involved
in this project.</h2>
<h1>Features</h1>
<ul>
<li>Newsletter Subscription</li>
<li>Quick Links for Social Media Pages</li>
<li>Embedded Google Maps Location of the Cafe</li>
<li></li>
</ul>
